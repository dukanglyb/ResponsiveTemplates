关于包的重要信息的文件夹，你就可以开始：

===================================================================================================================================================
- _documentation  		- 包含的管理和前端的主题一般的文档文件。

- _resources      		- 包含的商业授权和免费的照片和主题的PSD文件。

- theme       			- Metronic的HTML主题的主文件夹（默认版本）
	- assets        	- 包含主题相关的文件（CSS，JS，图像等）。
		- admin			- 包含的管理主题相关的文件（CSS，JS，图像等）
		- frontend   	- 包含的管理主题相关的文件（的CSS，爵士，图像等）。
		- global  		- 包含全局Metronic的框架组件，定制的第三方插件。
		              	  所有全局文件夹下的特征的在前端的主题和管理主题都可以使用。

	- sass				- 包含SASS（sass-lang.com）为admin主题（仅管理员默认版本完全支持SASS）的文件。
	- templates       	- 包含的管理和前端的主题模板页面。
		- admin       	- 包含的管理主题模板页面。
		- frontend    	- 包含前端主题模板的页面。

- theme_rtl   			- 为RTL Metronic的HTML主题的主文件夹（从右到左（RTL）书写的支持如：阿拉伯语，伊朗）。
					      该RTL版本包含的默认版本的所有文件进行语言切换（从LTR到RTL和RTL到LTR）

	- assets        	- 包含的主题相关的文件（的CSS，爵士，图像，插件等）。
		- admin			- 包含的管理主题相关的文件（CSS，JS，图像等）。
		- frontend   	- 包含的管理主题相关的文件（的CSS，爵士，图像等）。
		- global  		- 包含全局Metronic的框架成品的配件，定制的第三方插件。
		              	  所有全局文件夹下的特征的在前端的主题和管理主题都可以使用
	- templates       	- 包含的管理和前端的主题模板页面。
		- admin       	- 包含的管理主题模板页面。
		- admin2       	- 包含管理员2主题模板页面。
		- admin3       	- 包含管理3主题模板页面。
		- frontend    	- 包含前端主题模板的页面。
===================================================================================================================================================

重要提示：
前端为主题，包括所有的演示图像/照片在购买的主题文件夹仅供参考。
为了演示目的，我们使用来自http://photodune.net/购买了一些图像。
然而，如果你喜欢用这些photodune图像（在“资源/前端/企业/照片”上市“资源/前端/电子商务/照片”文件夹），那么你将
需要购买一个许可证为每个使用的图像。美创不会从photodune转让任何许可证的图像。这是不是在“资源/前端/企业/照片”，并列出任何图像“资源/前端/电子商务/照片”文件夹，可免费使用，因为它们是开放的源图像。
